# Expense Tracker — Task Description

Build a shared expense tracker where all authenticated users can view expenses submitted by anyone, but can only edit or delete their own. An admin account has full edit and delete access over all expenses.

## Features

- **View** — all users can browse the full expense list (amount, category, description, date, submitted by)
- **Create** — any authenticated user can submit a new expense
- **Edit / Delete** — users can only modify their own expenses; admins can modify any expense
- **Admin role** — a seeded admin account (`admin@example.com`) with elevated permissions

## Seed Data

- 1 admin user and 3 regular users
- ~10 sample expenses spread across users and categories (Food, Transport, Utilities, Entertainment, Other)
