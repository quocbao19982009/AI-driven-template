# Implementation Checklist

Use this when scaffolding a new feature to verify all layers are complete.

## Backend

- [ ] Entity created in `Features/[Name]/[Name].cs`
- [ ] DTOs created in `Features/[Name]/[Name]Dtos.cs`
- [ ] Validator created in `Features/[Name]/[Name]Validator.cs`
- [ ] Repository interface + implementation created
- [ ] Service interface + implementation created
- [ ] Controller created with all required endpoints
- [ ] Registered in `Program.cs`
- [ ] Migration created and applied

## API Sync

- [ ] `npm run api:sync` run from repo root
- [ ] `api/generated/[name]/` folder generated by Orval

## Frontend

- [ ] `features/[name]/` folder created with all layers
- [ ] UI matches the spec description
- [ ] Redux slice registered in `store/index.ts`
- [ ] Route added in `routes/[name]/index.tsx`
- [ ] Link added to navigation in `components/layout/app-layout.tsx`
- [ ] Translation keys added to both `en.json` and `fi.json`
