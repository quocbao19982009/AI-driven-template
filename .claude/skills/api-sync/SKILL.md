# API Sync Pipeline

Run the full API sync pipeline: build backend → export Swagger → generate Orval hooks.

## Steps

1. **Run the sync from the repo root:**

   ```bash
   npm run api:sync
   ```

2. **If the backend build fails:**
   - Report the build errors clearly
   - Do NOT proceed — the backend must build successfully before Orval can generate

3. **If successful, show what changed:**

   ```bash
   git diff --stat frontend/src/api/generated/
   ```

4. **Remind the user:**
   > Files under `frontend/src/api/generated/` are auto-generated by Orval. Never edit them manually — they are overwritten on every sync.
